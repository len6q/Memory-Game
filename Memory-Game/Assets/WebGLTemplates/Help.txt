Желаете зафиксировать экран с игрой в соотношений 16 на 9 или 9 на 16? 
Тогда наши шаблоны для вас.

● ChikasBridge: данный шаблон предназначен для тех, кто пользуется Instant Games Bridge - плагином для публикации веб-игр на разных платформах. 
● ChikasPluginYG: этот шаблон разработан специально для пользователей Plugin YG версии 1.4 - плагина для Яндекс.Игр. 
● ChikasFixed: если вы не используете какие-либо плагины, то этот шаблон для вас.
● ChikasFull: текущий шаблон не предназначен для фиксации экрана, он лишь адаптирует игру на весь экран.

Во всех шаблонах уже встроена инициализация SDK. Кроме того, в них также удален скролл и лонгтап на IOS.

Как использовать?
1. Для начала разархивируйте .zip файл и перекиньте папку WebGLTemplates в Unity.
2. После в Player Settings (File > Build Settings > PlayerSettings) выберите необходимый вам шаблон. 
3. Укажите соотношение сторон в Default Canvas Width и Height, и по желанию можете изменить цвет полос в Bar color.
